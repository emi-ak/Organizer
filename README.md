# Em's Planner

A pastel pink, glassmorphism planner website for an undergraduate student preparing for medical school.

## Updates included

- Renamed to **Em's Planner**
- Dashboard title simplified
- Removed reset demo button and intro subtitle
- Activity dates added
- Activities are editable after creation
- Timeline now includes **Extracurriculars**
- Timeline milestones are numbered and ordered by date
- Timeline milestones are editable
- Coursework changed from courses to modules
- Modules use Term 1 / Term 2 / Term 3
- Assignments can be added under module headings
- Assignment marks show UK classification
- Module totals are calculated using assignment weightings
- Overall academic average uses the average of each module total
- UK grade scale:
  - 1st Honors: 70%–100%
  - 2:1 Honors: 60%–69%
  - 2:2 Honors: 50%–59%
  - 3rd Honors: 40%–49%
- Prerequisite checkbox removed
- Vision board supports image upload, rearranging, and deleting
- Affirmations now have dates and are editable
- Resources category dropdown removed
- Goals now include due date, category, and editing
- Automatic saving with browser localStorage

## Files to upload to GitHub

## Preview locally

Unzip the folder and double-click `index.html`.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload all four files.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select branch: `main`.
6. Select folder: `/root`.
7. Save.
